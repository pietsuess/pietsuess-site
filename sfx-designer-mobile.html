SPACE = Play/Stop SHIFT+drag = Fine knob SHIFT+click = Multi-select CTRL/CMD+click through = Pop off ESC = Close help
CABLES

Drag from output jack to start cable. Drop on module or input jack to connect. Click input jack to disconnect. Cable style selector in top bar.

THROUGH NODES

Drag onto a cable to insert inline. Alt+drag to start cable from output. Ctrl/Cmd+click to pop off (reconnects cable). Double-click to toggle horizontal/vertical.

PATTERN BOARDS

4x4 grid = 16-step sequencer. Click squares to toggle steps on/off. Each output (1/2/3/4) has its own pattern + BPM. As effect: Generator > Pattern > Mix (gates audio). As trigger: Pattern out > Generator in (triggers hits). Set generator RATE to OFF when using pattern trigger.

AUTO PATCH

Drag modules below the shelf into columns. AUTO PATCH chains each column top-to-bottom. Columns detected by X proximity.
